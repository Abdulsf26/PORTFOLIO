# Personal Portfolio Website

Welcome to my personal portfolio! This repository contains the source code for my interactive and fully responsive portfolio, showcasing my skills, educational background, certifications, and previous work.

## 🚀 Live Demo

*You can view the live site here:* [Portfolio Live Demo](#) *(Update with your Vercel/Netlify link when deployed)*

## ✨ Features

- **Modern UI/UX:** Clean, elegant design with a dynamic light and dark theme functionality.
- **Smooth Animations:** Built with Framer Motion for beautiful page transitions, scroll effects, and hover animations.
- **Custom Components:** Styled-Components utilized for modular, dynamic, and scoped CSS.
- **Sections:**
  - **About:** A glimpse into who I am and what I love doing.
  - **Skills:** Detailed cards highlighting my technical stack and proficiencies.
  - **Work:** A gallery of my personal and client projects (SaaS apps, automated systems, etc.).
  - **Education:** A dedicated timeline displaying my academic journey.
  - **Certifications:** Awards and recognizable certifications I've achieved.
  - **Blog:** Highlights of my recent events, achievements, and tech explorations.
- **Particle Backgrounds:** Interactive particle JS background for an immersive feel.

## 🛠️ Built With

- **[React 19](https://react.dev/)** - Component-based UI library
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **[Styled-Components](https://styled-components.com/)** - CSS-in-JS styling
- **[Motion (Framer Motion)](https://motion.dev/)** - Animation library for React
- **[React Router DOM](https://reactrouter.com/)** - Client-side routing

## 📦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need Node.js and npm (or bun/yarn) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdulsf26/PORTFOLIO.git
   ```
2. Navigate into the directory:
   ```bash
   cd PORTFOLIO
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
